# liuliu

学习通六六助手 userscript 仓库。

## 功能

- OpenAI 接口答题辅助
- 自动识别题型并生成建议答案
- 可选自动勾选答案
- 最小化悬浮图标

## 文件

- `学习通六六助手[AI答题][一键启动][最小化运行]-3.3.4.user.js`: 主脚本

## 使用

1. 安装支持 UserScript 的浏览器扩展，例如 Tampermonkey。
2. 导入仓库中的 `.user.js` 脚本。
3. 在脚本设置面板中填写 API Key、Base URL 和模型名称。
4. 打开学习通题目页面后使用面板进行答题辅助。
